About My HTTP Server

This website is served by a custom HTTP server written in Java. The server handles HTTP requests, parses request lines and headers, routes endpoints, and generates responses without using any web framework.

Features

  • Custom HTTP request parser
  • Routing system for endpoints
  • Special endpoints like /httpbin and /video
  • Static file serving
  • Multithreaded request handling

This project was built as a learning exercise to better understand how web servers, HTTP requests, and networking work under the hood.
